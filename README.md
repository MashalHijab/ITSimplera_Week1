# Week 1 - Networking Fundamentals & Simulation Tools

## Overview
This project covers basic networking concepts and hands-on LAN topology 
creation using Cisco Packet Tracer and GNS3.

## Contents
- Report.pdf - Full documentation
- Cisco_Packet_Tracer.pkt - Packet Tracer topology file
- GNS3_Project - GNS3 project files
- Screenshots - Topology, IP config, and ping test results

## Topology
Simple LAN with 1 switch and 3 PCs, static IP addressing (192.168.1.0/24), 
connectivity verified via ping.

## Tools Used
- Cisco Packet Tracer
- GNS3
- PuTTY
